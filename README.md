# VUE-PROJECT-TEMPLATE

Vue 项目模板，按分支提供不同版本，选择对应分支克隆即可。

## 快速开始

```bash
# VUE2
git clone -b VUE2 https://github.com/IBINIBIN/VUE-PROJECT-TEMPLATE.git VUE2-PROJECT-TEMPLATE

# VUE3-JS
git clone -b VUE3-JS https://github.com/IBINIBIN/VUE-PROJECT-TEMPLATE.git VUE3-JS-PROJECT-TEMPLATE

# VUE3-TS
git clone -b VUE3-TS https://github.com/IBINIBIN/VUE-PROJECT-TEMPLATE.git VUE3-TS-PROJECT-TEMPLATE

```
